## 启动方式

在终端执行（需在 `frontend-html` 目录下）：

```bash
cd "/Users/fangdanyang/Desktop/HUAXIA BANK-CLIMATE RISK/climate-risk-stress-test/frontend-html"
python3 -m http.server 8765
```

保持该终端窗口不要关闭。若 8765 端口已被占用，可改成其他端口，例如 `8888`，并把下面链接里的 `8765` 一并替换。
