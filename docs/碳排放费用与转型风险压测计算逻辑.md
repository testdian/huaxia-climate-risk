# 高碳行业碳排放费用与转型风险压力测试计算逻辑整理

> 依据扫描版《高碳行业碳排放费用计算方法》《转型风险压力测试情景设置》《高碳行业分类表》整理

## 一、碳排放费用计算逻辑

### 1. 有碳排放数据企业

| 环节 | 计算公式 / 处理规则 |
|------|---------------------|
| 碳排放因子 | 碳排放因子 = 基期碳排放量 ÷ 营业收入。基期碳排放量单位为吨二氧化碳当量，营业收入单位为百万元，因子单位为吨二氧化碳当量/百万元。 |
| t期碳排放量 | 假设测试期企业不提升生产工艺、单位产品碳排放量保持不变：碳排放量_t = 营业收入_t × 碳排放因子。 |
| 碳排放费用 | 碳排放费用_t = 碳排放量_t ×（1 - 免费配额比例_t）× 碳价_t。 |

### 2. 无企业碳排放数据或需按行业估算的企业

## 二、转型风险压力测试情景与关键假设

| 环节 | 计算公式 / 处理规则 |
|------|---------------------|
| 情景设置 | 设置1个基准情景和2个转型情景。基准情景为维持现有政策不变；转型情景为「温室世界」和「有序转型」。 |
| 测试期限 | 以2025年末为基期，测试期至2040年。银行资产负债结构按静态口径处理，即测试期间银行资产负债保持不变。 |
| 免费配额比例 | 假设高碳企业碳排放免费配额比例在给定时间区间内线性变化。 |
| 议价能力 | 所有企业均不具备议价能力，高碳企业碳排放成本无法转嫁。 |

### 测试情景免费配额比例

| 测试情景 | 2025年 | 2040年 |
|----------|--------|--------|
| 现有政策 | 100% | 75% |
| 温室世界 | 100% | 85% |
| 有序转型 | 100% | 55% |

### 3. CCUS 技术使用假设

| 可用行业 | 使用条件 | 碳排放成本 |
|----------|----------|------------|
| 火电、水泥、石化、造纸、金属冶炼、平板玻璃制造 | 2024年末净利润为正 | 若T期碳价 < 500元/吨：成本 =（1 - 免费配额比例）× T期排放量 × 碳价；若T期碳价 ≥ 500元/吨：成本 =（1 - 免费配额比例）× T期排放量 × 500。 |

## 三、对公贷款压力测试计算链条

测试覆盖参试银行对公贷款和个人贷款；对公贷款进一步划分为高碳行业贷款和其他行业贷款。高碳行业包括火电（含电力供应）、钢铁、建材、石化、化工、造纸、航空（含机场）和有色金属冶炼等。

| 计算环节 | 规则 |
|----------|------|
| 营业收入 | 营业收入_t = 营业收入_{t-1} ×（1 + 行业增加值增速_t）。原文为逐期更新口径，按上下文整理为前一期滚动计算。 |
| 营业支出：高碳行业 | 营业支出_t = 营业收入_t × 基期成本收入比 + 碳排放费用_t。 |
| 营业支出：其他行业 | 营业支出_t = 营业收入_t × 基期成本收入比。 |
| 利润总额 | 利润总额_t = 营业收入_t - 营业支出_t。 |
| 净利润 | 若利润总额_t > 0，企业缴纳所得税，净利润_t = 利润总额_t × 75%；否则净利润_t = 利润总额_t。 |
| 资产端调整 | 净利润为正：按基期比例分配至货币资金、应收票据、应收账款和存货。净利润为负：先抵扣货币资金；货币资金为0后抵扣其他流动资产；流动资产为0后抵扣固定资产，固定资产折价系数为50%。 |
| 负债端调整 | 负债端不发生变化。 |
| 所有者权益 | 当期净利润全部计入留存收益；如净利润为负且需抵扣固定资产，固定资产折价损失在留存收益中一并扣除。 |

### 违约判定

| 条件 | 规则 |
|------|------|
| 基期资产负债率低于100% | 测试过程中资产负债率超过100%，判定违约，对应贷款计入不良贷款。 |
| 基期资产负债率大于100% | 测试过程中资产负债率增幅超过20%，判定违约，对应贷款计入不良贷款。 |
| 资产总计等于0 | 判定违约，对应贷款计入不良贷款。 |
